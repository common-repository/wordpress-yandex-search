Adds Yandex search widget to your blog. 

== Description ==

Tiny plug-in to add Yandex search functionality to your blog.
Uses JavaScript.

== Installation ==

1. Upload wp-yandex-search.php to plugins directory
2. Activate the plugin in 'Plugins' menu of wp-admin
3. Add a widget to sidebar. That's it!
